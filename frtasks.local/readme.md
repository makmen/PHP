# Проект task manager - веб-приложение для управления проектами и задачами (в том числе для описания ошибок). Проект разработан на собственном движке. 
Версия php: 7.0

Каждая задача в системе имеет свой статус. Определенные статусы могут менять только пользователи, имеющие на это привилегии.  Трекеры являются основной классификацией, по которой сортируются задачи в проекте. У каждой задачи в обязательном порядке есть описание и автор, приоритет, также задача привязана к трекеру и параметр «Назначена», который на начальном этапе допускается пустым. При обновлении истории задачи Менеджерам разрешается менять все поля у задачи, Исполнителю и Тестеру только комментарий и статус.

Проект может быть добавлен менеджером или администратором. Благодаря этой сущности возможно организовать совместную работу над задачами в рамках одного проекта или нескольких.

За каждым пользователем закреплена конкретная роль (администратор, менеджер, разработчик, тестировщик, дополнительный пользователь). Каждой роли назначены свои свойства, ограничивающие возможности пользователей.
